# Preface to the Publication Edition

This edition is designed as a professional reference and advanced teaching text. It combines platform architecture, implementation analysis, code listings, operational design, and course-oriented material in a single volume.

The body of the book is maintained in the repository manuscript and is intended to be exportable to EPUB, HTML, and other long-form formats through Pandoc.

---

# About This Edition

This publication edition is intended for:

- advanced data engineers,
- analytics engineers,
- technical instructors,
- architects designing small-team or budget-conscious platforms,
- and practitioners evaluating local-first analytical runtimes with cloud lakehouse storage.

It is not a beginner’s introduction. It is a working handbook built around an actual implementation.

---

# Bibliographic Note

This book is intentionally practice-led. Its bibliography combines platform documentation, implementation artifacts from the repository, and engineering patterns derived from working code. Readers should treat the repository and the exported edition as companion references.
